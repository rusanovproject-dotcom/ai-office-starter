## Для Демиурга — как развернуть этот шаблон (не показывать человеку)

**Строка в карту команды** `team/map.md` (формат — как у остальных):

```
- **<Имя>** (<роль кратко>) — <за что отвечает>, где живёт → team/agents/<slug>/core.md
```

**Строка `route:`** во frontmatter скилла — только если у сотрудника есть скилл:

```
route: "<ключ1> | <ключ2> => <что делает одной фразой>"
```
