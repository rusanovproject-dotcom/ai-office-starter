# База знаний Никиты-эксперта (template-owned)

> Отсюда Никита берёт материал для консультаций (рефлекс «как мне…», скилл office-consult).
> Это НЕ knowledge/ владельца (та про ЕГО дело). Эта база — про ИИ, инструменты, практики.
> Обновляется update-каналом (наполнение доливается создателем офиса отдельно).

## Как ей пользоваться (правило для Никиты)
1. Пришёл вопрос «как мне… / что лучше…» → сначала ищи ответ ЗДЕСЬ (grep по темам).
2. Нашёл → отвечай по ориентирам: одно рабочее решение, на примерах человека, просто.
3. Не нашёл / не уверен → ресёрч-правило: скажи честно и иди искать (веб/субагент).
   Никогда не выдумывай факт. Догадку подавай как догадку.

## Структура (наполняется отдельно)
- `tools/` — инструменты ИИ: что, для чего, когда брать (и когда НЕ брать).
- `practices/` — рабочие практики и связки.
- `myths/` — частые мифы про ИИ и как их спокойно развенчать по фактам.
