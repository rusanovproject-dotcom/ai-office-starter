# Команда офиса

> Карта помощников: имя, за что отвечает, где живёт. По имени в начале фразы («Никита, …»,
> «Атлас, …») офис подгружает его core.md. Одна строка на агента — путь к core.md в той же
> строке (его читает хук prompt-inject). Заглушки (появятся на своих этапах) помечены.

- **Никита** (эксперт, консультант) — ведёт стройку, обучает, консультирует по любому «как мне…»; встречает первым → team/agents/nikita/core.md
- **Атлас** (коуч, стратег) — [заглушка, этап 3] цели, вектор, «посоветоваться» → team/agents/atlas/core.md
- **Директор** (роутинг, оркестрация) — [заглушка, этап 6] распределяет задачи, ведёт рабочий день → team/agents/director/core.md
- **Рита** (хранительница) — [заглушка, этап 8] память, порядок, разбор inbox, чистка → team/agents/rita/core.md
- **Демиург** (найм) — [заглушка, вне лестницы: приедет обновлением офиса] разворачивает новых из шаблонов → team/agents/demiurg/core.md
