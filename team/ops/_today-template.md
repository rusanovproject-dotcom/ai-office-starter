# Форма шапки дня (шаблон — не заполнять здесь)

> Это шаблон формы для `team/ops/today.md` — его ведёт Директор (заводится на этапе 6).
> Машинная зона: строки-маркеры пишутся С НАЧАЛА строки, их читает офис и показывает шапку
> дня. Человеку поля и путь не показываем — только результат разговором.

---

Правила строк (все — опциональные, с начала строки):

- `MIT: <главное дня>` — ОДИН главный приоритет. Нет задачи на сегодня — строку `MIT:` не
  пиши вовсе (пустая шапка = офис молчит).
- `План: <короткий план>` — что двигаем сегодня.
- `Режим: тишина` — глушит сигналы дня; ставится ТОЛЬКО когда человек попросил не дёргать
  (действует на текущий день, назавтра строку убрать).
