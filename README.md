# Твой AI-офис

Это твой личный офис — команда цифровых сотрудников, которая помнит про тебя и твоё дело.
Он собирается под тебя вместе с Никитой-экспертом: просто запусти и напиши что угодно.

Как начать и что будет по шагам — смотри в гайде (дорожная карта этапов).
Внутри всё объяснит Никита. Техническое — на нём, твоё дело — рассказывать о себе.
