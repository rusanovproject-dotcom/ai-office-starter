# AI Office Starter

Каркас AI-офиса, который собирается под тебя. Не заполненный шаблон — **конструктор с ведущим**: агент-Эксперт принимает твои данные, объясняет каждый шаг и разворачивает офис вместе с тобой.

> 🚧 **v3 в сборке.** Скелет наполняется — следи за обновлениями.

## Как начать

1. Нажми **«Use this template»** и создай свою копию.
2. Открой её в Claude Code.
3. Следуй гайду — Эксперт встретит тебя и поведёт по шагам.

## Что внутри

- Ядро офиса: хуки, защитный контур, канал обновлений.
- Агент-Эксперт — строит офис под твоё дело.
- Штат: коуч, директор, хранитель памяти — появляются по ходу стройки.

Твой офис — приватное место. Он живёт у тебя и никуда не публикуется.
